# 🦉 WordOwl

Умная мини-игра для изучения английских слов. Чистый статический веб (без сборки),
работает прямо в браузере и на GitHub Pages.

В основе — **интервальное повторение FSRS** (тот самый алгоритм, на который Anki
перешёл в 2023): каждое слово показывается ровно в тот момент, когда ты вот-вот
его забудешь. Это в разы эффективнее обычной зубрёжки.

## Что внутри

- **3000 самых частотных слов** EN→RU, отсортированы по полезности и разбиты на
  уровни **A1 → C1**. Учишь сначала то, что реально встречается.
- **4 режима тренировки:**
  - 🃏 **Карточки** — вспомни и оцени себя (Again / Hard / Good / Easy)
  - ✅ **Выбор** — выбери правильный перевод (RU↔EN, в обе стороны)
  - ⌨️ **Ввод** — напиши слово по-английски (допускается опечатка в 1 букву)
  - 🎧 **Аудирование** — запиши на слух
  - 🧠 **Умный микс** — режим сам подбирает упражнение по «зрелости» карточки
- **Озвучка** — настоящее аудио из [dictionaryapi.dev](https://dictionaryapi.dev)
  (без ключа), фолбэк на синтез речи браузера.
- **Картинки** — фото из [Pixabay](https://pixabay.com/api/docs/) + emoji-фолбэк.
- **Транскрипция (IPA), определение и пример** подтягиваются автоматически и кэшируются.
- **Геймификация** — XP, уровни, ежедневная серия (streak), цель «новых слов в день»,
  конфетти за идеальные сессии.
- **Статистика** — прогресс по уровням, активность за 14 дней, точность, закреплённые слова.
- **Словарь** — поиск и просмотр всех слов с озвучкой.
- Прогресс хранится локально (localStorage), есть экспорт/импорт.

## Запуск

Из-за ES-модулей нужен http-сервер (не `file://`):

```bash
# любой статический сервер, например:
python3 -m http.server 8000
# затем открой http://localhost:8000
```

Либо просто включи **GitHub Pages** (Settings → Pages → ветка `main`, корень `/`).

## Картинки (Pixabay)

Дефолтный ключ уже вшит. Свой бесплатный ключ можно получить на
<https://pixabay.com/api/docs/> и вставить в **Настройки → Pixabay API-ключ**.
Без ключа картинки заменяются на emoji.

## Данные

Словарь (`js/data.words.js`) генерируется из открытых источников:

- частотность + части речи — [filiph/english_words](https://github.com/filiph/english_words) (COCA top-5000)
- переводы — [MUSE en-ru](https://github.com/facebookresearch/MUSE) (53k пар)
- emoji — `tools/emoji-map.json` (вручную)

Пересобрать (нужен интернет):

```bash
npm run build-data        # = node tools/build-dataset.mjs [maxWords]
```

## Тесты

```bash
npm install               # ставит jsdom (dev)
npm test                  # headless-смоук всех экранов и сессии
```

FSRS-планировщик в `js/srs.js` реализован по
[FSRS-4.5](https://github.com/open-spaced-repetition/fsrs4anki/wiki) со
стандартными весами.

## Структура

```
index.html              точка входа
css/style.css           стили (тёмная/светлая тема)
js/
  app.js                оркестратор: экраны, сессия, геймификация
  srs.js                FSRS-планировщик
  state.js              localStorage: прогресс, настройки, статистика
  data.js               очередь обучения, дистракторы, dictionaryapi.dev
  data.words.js         сгенерированный словарь (3000 слов)
  audio.js              озвучка
  images.js             Pixabay + emoji + плейсхолдер
  ui.js                 DOM-хелперы
  games/                flashcard · choice · typing · listening
tools/
  build-dataset.mjs     сборка словаря из открытых источников
  emoji-map.json        слово → emoji
  smoke.mjs             headless-тест
```
